Matching Reveal Game

A simple memory-matching game built with HTML, CSS, and JavaScript. Match pairs of food emoji tiles to reveal a secret message.

Features
- Interactive tile-matching gameplay
- Satisfying animations and sound effects
- Customizable secret message reveal
- Responsive design that works on various screen sizes
- Beautiful visual styling with gradient tiles and smooth hover effects

How to Play
1. Click the "Start Game" button to begin
2. Click on tiles to reveal the emoji hidden underneath
3. Try to find matching pairs of emojis
4. When you match a pair, they stay revealed
5. If the tiles don't match, they flip back over
6. Match all pairs to reveal the secret message!

Technical Details
The game uses:
- Custom CSS animations for a polished feel
- Google Fonts 
- Local sound effects for audio feedback
- JavaScript for game logic and randomization

Setup
Simply download all files to a directory and open the HTML file in a modern web browser. Make sure the image file ("eggs.jpg") and sound files ("mouse-click-sound-233951.mp3" and "bubblepop-254773.mp3") are in the same directory as the HTML file.

Customization
You can easily customize:
- The background image by replacing "eggs.jpg"
- The secret message by editing the content of the messageContent` div
- The sound effects by replacing the MP3 files
- The tile emojis by modifying the "symbols" array in the JavaScript
